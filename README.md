# DishNest

DishNest is a web application that allows users to browse a variety of recipes in one place. Users can explore different dishes, view ingredients, follow cooking instructions, and save their favorite recipes for easy access later. The goal of DishNest is to make discovering and organizing recipes simple and convenient.

## Features

* Browse recipes through a clean and simple interface
* View ingredients and step-by-step cooking instructions
* Favorite recipes to easily access them later
* Organized recipe pages for different dishes
* Leave reviews and rating on recipes
* Search through recipe using search engine
* Login and signup system
  

## Technologies Used

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Node.js
* Express.js
* SQLite3


## How to Run the Application

Follow these steps to run the DishNest web application locally.

### 1. Install Node.js

Make sure Node.js is installed on your computer. You can check by running:

```
node -v
```

### 2. Navigate to the Backend Folder

Open a terminal in the project directory and move into the backend folder:

```
cd backend
```

### 3. Install Dependencies

Install the required backend packages:

```
npm install
```

### 4. Start the Server

Run the server with:

```
node server.js
```

If the server starts successfully, you should see a message in the terminal indicating that the server is running.

### 5. Open the Frontend

Open the frontend files in your browser (for example, open `login.html`). The frontend will connect to the backend server to retrieve and manage recipe data.
